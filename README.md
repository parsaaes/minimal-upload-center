# Minimal Upload Center
Spin up a quick upload center for basic use cases.

## Prerequisites
- Golang (Only if you want to build it yourself)

## Quick Start

### Build it yourself
```bash
go run main.go
```

Or
### Download the pre-built version from releases
```bash
chmod +x minimal-upload-center-darwin-arm64
./minimal-upload-center-darwin-arm64
```
The UI is ready on `:7677`.
